  ## Description
  This command-line application manages a company's employee database, using Node.js, Inquirer, and MySQL. 
  
  ## Installation
  - Run 
    * `npm install inquirer`
    * `npm install mysql2`

  ## Usage
  Open the index.js file. In Terminal, type `node index.js`. Press Return.
